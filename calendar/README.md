# Calendar App

A calendar application built with vanilla JavaScript that allows users
to create, edit, and delete events directly from the interface.

## Features
- Month navigation
- Add events with name, time, and location
- Edit existing events
- Delete events
- Data persistence using localStorage
- Responsive interface

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage

## Usage
1. Click on a day in the calendar
2. Fill in the event details
3. Save or delete the event as needed

Events are automatically saved in the browser.
