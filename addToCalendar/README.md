# Add To Calendar

A web tool that allows users to generate downloadable calendar events
to import into external calendars such as Google Calendar, Apple Calendar,
or Outlook.

## Features
- Calendar event generation
- Downloadable event files
- Simple and intuitive interface
- Support for multiple calendar platforms

## Technologies
- HTML5
- CSS3
- JavaScript

## Usage
1. Fill in the event details
2. Download the generated file
3. Import the file into your preferred calendar application
